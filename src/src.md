# src 技术总结

## 职责定位

`frontend/src/` 是前端源码根目录，承载路由、页面、布局、组件、样式、工具函数与共享 API 封装。

## 分层结构

| 子目录 | 作用 |
| --- | --- |
| `components/` | 通用组件与编辑器组件 |
| `composables/` | 组合式逻辑 |
| `layouts/` | 页面外壳布局 |
| `pages/` | 路由页面 |
| `router/` | 路由定义与守卫 |
| `shared/` | API 客户端与模块映射 |
| `styles/` | 全局主题样式 |
| `types/` | 类型声明补充 |
| `utils/` | 差异分析与文本处理工具 |

## 技术要点

- 目录划分清晰，符合 Vue 3 中后台项目常见结构。
- 业务页面和公共逻辑拆分较彻底，便于后续在多个模块之间复用。
- 源码根目录同时承担应用入口、模块切换与通用 API 网关三类职责。
