# router 技术总结

## 职责定位

`frontend/src/router/` 负责前端路由定义、懒加载页面挂载以及简单登录守卫。

## 当前实现

| 文件 | 作用 |
| --- | --- |
| `index.ts` | 路由表、`AppLayout` 嵌套结构、未登录跳转和首次入口逻辑 |

## 技术要点

- 业务页统一挂在 `/` 下，并以 `AppLayout` 作为壳层。
- 路由按采集、翻译、导入导出、系统管理四类分组，和后端 API 边界基本对应。
- 守卫通过 `authToken` 与 `needSetPassword` 控制访问，逻辑简单直接。
- 页面均采用动态导入，减少首屏包体积。
