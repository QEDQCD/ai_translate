# shared 技术总结

## 职责定位

`frontend/src/shared/` 提供跨页面共享能力，当前包括统一 API 客户端、模块映射表与辅助函数。

## 主要文件

| 文件 | 作用 |
| --- | --- |
| `api.ts` | Axios 实例、鉴权头、租户头、错误拦截、按业务分组的 API 封装 |
| `moduleMapping.ts` | 页面与模块的映射、菜单结构、权限过滤 |
| `utils.ts` | 通用前端辅助方法 |

## 技术要点

- `api.ts` 是前端与后端交互的网关层，统一处理 Token、`X-Tenant-ID`、错误提示和长超时请求。
- `moduleMapping.ts` 把菜单、默认路由、页面归属和权限控制集中配置，降低导航散落风险。
- 共享层设计较成熟，是整个前端的核心基础设施之一。
